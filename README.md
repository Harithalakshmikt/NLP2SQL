# NLP to SQL System

## ✅ Overview
This project converts natural language questions into SQL queries using an LLM and executes them on a SQLite database.

## ⚙️ Tech Stack
- FastAPI
- SQLite
- Groq (LLM - llama3)
- Python

## 🔑 LLM Used
Groq API with model: llama-3.3-70b-versatile
